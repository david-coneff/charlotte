Self-Contained Domain Crawler

Crawls one domain's internal links, records first-tier external links, and reports here — all in this single page.

Local CORS proxy — required when opening this file via file://

A file:// page can't open a network socket, so it can't host its own proxy. This page carries the proxy instead: download it, run it with Node, and point the proxy field above at it. It binds to 127.0.0.1 only and needs no npm install.

  1. Click Download proxy (saves local-cors-proxy.js).
  2. In a terminal: node local-cors-proxy.js
  3. Leave the proxy field set to http://127.0.0.1:8080/?url={url} and start the crawl.
